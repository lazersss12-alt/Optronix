# Official Website of Candid Optronix

Welcome to the official repository for the website of **Candid Optronix**! This repository contains the source code and resources for our website hosted at [optronix.in](https://www.optronix.in/).

---

## 🚀 About Candid Optronix

At **Candid Optronix**, we are industry leaders specializing in the manufacturing of **optical fiber cables**, **surface mounting technology (SMT) lines for electronics**, and distributing state-of-the-art testing and measurement equipment. Our mission is to provide innovative, reliable, and high-quality solutions to meet the demands of a fast-evolving industry.

---

## 🌟 Features

- **About Us**: Discover our company, mission, and values.
- **Products/Services**: Browse through our range of cutting-edge products and services.
- **Contact Us**: Reach out to our team for support or inquiries.
- **Blog/News**: Stay updated with the latest industry insights and company news.

## 🛠️ Technologies Used

The website is built using the following technologies:

- **Frontend**: React.js, HTML, CSS, JavaScript
- **Styling**: CSS Modules and Global CSS
- **Hosting**: GitHub Pages
